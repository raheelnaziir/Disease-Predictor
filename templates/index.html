This tool uses a Random Forest machine learning model trained on a dataset of 41 diseases and ~130 symptoms.


It combines ML prediction with clinical post-filtering based on onset, episode history, and neurological deficit for improved accuracy.


Built with Python · Flask · scikit-learn · HTML/CSS/JS


⚠️ For educational purposes only. Not a substitute for professional medical advice.

How to use

  • Type a symptom in the input box and select from suggestions
  • Add multiple symptoms for better accuracy
  • Fill in the clinical dropdowns for improved results
  • Press send or hit Enter to get your prediction

Tips

  • Add at least 3–4 symptoms for best accuracy
  • Use "Add manually" if your symptom isn't in the list
  • The confidence bar shows how certain the model is

⚠️ Always consult a real doctor for medical decisions.
